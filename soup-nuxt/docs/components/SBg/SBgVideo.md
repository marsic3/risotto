# SBgVideo

> Background video component

## Props

| Prop name   | Description                    | Type    | Values | Default |
| ----------- | ------------------------------ | ------- | ------ | ------- |
| video       | Video's object to play (.MP4)  | object  | -      |         |
| transparent | Adds transparency to the video | boolean | -      | false   |

---

```vue live
<SBgVideo :video="{}" />
```
