# SGallerySlider

> Gallery slider with main photo and small thumbnails at the bottom.

## Props

| Prop name | Description                        | Type  | Values | Default |
| --------- | ---------------------------------- | ----- | ------ | ------- |
| items     | Items to display in gallery slider | array | -      | []      |

---

```vue live
<SGallerySlider :items="[1, 2, 3]" />
```
