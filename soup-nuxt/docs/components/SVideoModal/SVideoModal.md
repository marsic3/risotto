# SVideoModal

> Modal with YouTube video iframe

## Props

| Prop name | Description         | Type    | Values | Default |
| --------- | ------------------- | ------- | ------ | ------- |
| show      | Shows the modal     | boolean | -      | false   |
| videoUrl  | Youtube's video URL | string  | -      |         |

## Events

| Event name | Type | Description |
| ---------- | ---- | ----------- |
| hidden     |      |
