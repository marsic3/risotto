# SNavigationItem

> Main navigation desktop.

## Props

| Prop name | Description     | Type   | Values              | Default |
| --------- | --------------- | ------ | ------------------- | ------- |
| item      | Item's object   | object | -                   |         |
| type      | Navigation type | string | `desktop`, `mobile` |         |

---

```vue live
<SNavigationItem :item="{}" type="desktop" />
```
